# Projeto-Bootstrap

Uma página de site de casas planejadas desenvolvida com HTML5, CSS3 e o framework Bootstrap.

---

## Descrição

Este projeto apresenta uma **página de site** voltada para casas planejadas, construída com foco em boas práticas de HTML5 e CSS3, integrando o framework Bootstrap para facilitar o layout responsivo, componentes e estilo.  
A página tem como objetivo servir de modelo ou base para sites do segmento imobiliário ou de arquitetura, podendo ser adaptada conforme necessidade.

---

## Características

- Layout responsivo para diferentes tamanhos de tela (desktop, tablet, celular).  
- Uso de componentes do Bootstrap (navbar, grid system, cards, etc.).  
- Estrutura limpa de HTML e CSS, facilitando manutenção e extensão.  
- Imagens ilustrativas de casas planejadas para apresentar portfólio ou galeria.  
- Boa separação entre estrutura (HTML), estilo (CSS) e recursos (imagens).

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- Bootstrap (versão conforme arquivo local ou CDN)  
- (Opcional) Imagens de exemplo no diretório `img/`  
- Navegadores modernos como alvo (Chrome, Firefox, Edge, Safari etc.)
